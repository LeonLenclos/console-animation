# console-animation

Make ascii-art animations in the console. (   ' _ >')


## Usage

You can run

```
./anim.py anims/face1.animtxt
```

to see an example

## .animtxt format

it is just a plain text file with all the frames separated by `#`.

eg.

```
#
(  ' _>')
#
(  ꙾ _>꙾)
#
(  ' _>')
#
...
```
